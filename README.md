# Python Game Alien_invasion
A simple plane shoot game written in Python
## Introduction
This project is written by Python & Pygame as an exercise to learn Python & Pygame lib
## Requirement
1. Python 3.6
2. Pygame 1.9.3
## How to Start Game
& python alien_invasion.py
